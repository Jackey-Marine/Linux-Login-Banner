# Linux Login Banner

## Compatibility
Login Banner tested working with Ubuntu 20.04.  
  
## Setup
Copy the contents inside *profile.txt* to */etc/profile* of your server.  

```bash
nano /etc/profile
```
  
***Paste*** and self-adjustment.
